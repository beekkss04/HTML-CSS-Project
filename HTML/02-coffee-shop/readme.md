# Coffee Shop Website ☕

This project is a simple coffee shop website built using **HTML and internal CSS**.
It demonstrates the use of **semantic HTML tags** and basic webpage layout.

---

## Project Goal

The purpose of this project is to practice semantic HTML structure and create a small themed website.

---

## Semantic Tags Used

* header
* nav
* main
* section
* article
* aside
* footer

These tags help organize the webpage structure clearly.

---

## Features

* Coffee shop landing page
* Navigation menu
* About section
* Menu items
* Special offer section
* Contact information
* Hero banner image
* Clean responsive layout

---

## Technologies Used

* HTML5
* CSS (Internal CSS)

---

## How to Run

1. Open the project folder.
2. Double click **index.html**
3. The website will open in your browser.
