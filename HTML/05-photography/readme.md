# Photography Portfolio (Black & White Theme)

This project is a photography portfolio website designed with a dark black & white theme.
It showcases a fictional photographer's work, services, and contact information.

---

## Project Objective

The objective of this project is to create a visually appealing photography website inspired by professional portfolio designs while practicing HTML layout and CSS styling.

---

## Features

* Dark photography-themed design
* Hero banner section
* About the photographer
* Photography services
* Portfolio gallery
* Contact form
* Responsive layout
* Black & white image style

---

## Technologies Used

* HTML5
* CSS (Internal Styling)

---

## Images Used

The project uses the following images placed in the same folder as the HTML file:

* hero.jpg
* photographer.jpg
* p1.jpg
* p2.jpg
* p3.jpg
* p4.jpg

These images represent different photography styles such as portraits, weddings, landscapes, and events.

## How to Run

1. Open the project folder.
2. Double-click **index.html**.
3. The photography portfolio website will open in your browser.
