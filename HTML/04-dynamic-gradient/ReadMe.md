# Dynamic Gradient Website

This project is a dynamic landing page inspired by the Notion website design.
The background gradient changes automatically using JavaScript, creating a modern and interactive effect.

---

## Project Objective

The goal of this project is to create a dynamic webpage where the background gradient continuously changes, similar to modern landing pages.

---

## Features

* Clean landing page layout
* Navigation bar
* Hero section with call-to-action buttons
* Animated gradient background
* Smooth transitions
* Responsive centered design

---

## Technologies Used

* HTML5
* CSS (Internal Styling)
* JavaScript

---

## How It Works

JavaScript is used to cycle through different gradient colors.

Every few seconds the background updates automatically, creating a dynamic visual effect.

## How to Run

1. Open the project folder.
2. Double click **index.html**
3. The webpage will open in your browser and the gradient animation will start automatically.
