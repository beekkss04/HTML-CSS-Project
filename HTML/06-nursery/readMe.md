# Green Garden Nursery Website 🌿

This project is a simple nursery website with a green theme and a flower gallery.
It showcases different flowers and basic nursery services using HTML and internal CSS.

---

## Project Objective

The goal of this project is to design a nature-themed website that displays plants and flowers in a gallery layout while practicing webpage structure and styling.

---

## Features

* Green themed design
* Hero banner with garden image
* About the nursery section
* Flower gallery
* Nursery services section
* Contact form
* Simple and clean layout

---

## Technologies Used

* HTML5
* CSS (Internal Styling)
