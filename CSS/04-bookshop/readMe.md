# BookNest – Bookshop Website 📚

This project is a simple **Bookshop website** created using HTML and internal CSS.
It displays popular books, categories, and a basic contact section.

---

## Project Objective

The objective of this project is to design a basic e-commerce style webpage for a bookstore and practice webpage layout, images, and styling using HTML and CSS.

---

## Features

* Navigation bar
* Hero banner with books background
* Popular books section
* Book cards with images
* Categories section
* Contact form
* Footer

---

## Technologies Used

* HTML5
* CSS (Internal Styling)

---

## Images Used

The following images are used in this project:

* books.jpeg (hero background)
* b1.jpg
* b2.jpeg
* b3.jpg

These images represent book covers and a bookstore background.

1. Open the project folder.
2. Double-click **book.html**
3. The Bookshop website will open in your browser.
