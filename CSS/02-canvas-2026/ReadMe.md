# Welcome to 2026 – Canvas Animation

This project demonstrates the use of the **HTML Canvas element** to create graphics and animations using JavaScript.

The page displays an animated background with floating particles and a message welcoming the year **2026**.

---

## Project Objective

The objective of this project is to understand how graphics can be drawn dynamically in the browser using the Canvas API and JavaScript.

---

## Features

* Full-screen canvas
* Gradient animated background
* Moving particle animation
* Text overlay ("Welcome to 2026")
* Smooth animation using `requestAnimationFrame`

---

## Technologies Used

* HTML5
* CSS
* JavaScript
* Canvas API

---

## Concepts Demonstrated

This project demonstrates:

* `<canvas>` element
* JavaScript drawing using `getContext("2d")`
* Gradients
* Animation loops
* Dynamic rendering

---

## Example Code

Drawing a circle particle:

```javascript
ctx.beginPath();
ctx.arc(x, y, size, 0, Math.PI * 2);
ctx.fill();
```

This creates circular particles that move across the screen.
