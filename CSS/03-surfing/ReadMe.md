# WaveRiders – Surfing Website

This project is a simple surfing-themed website created using HTML and internal CSS.
It represents a fictional surfing academy called **WaveRiders Surf Academy**.

---

## Project Objective

The objective of this project is to design a themed webpage inspired by surfing culture and ocean sports while practicing layout, images, and styling with HTML and CSS.

---

## Features

* Surf-themed navigation bar
* Hero section with background image
* Call-to-action button
* About section
* Course cards
* Surfing image gallery
* Footer section

---

## Technologies Used

* HTML5
* CSS (Internal Styling)

---

## Images Used

The following images are used in this project:

* surf-hero.jpg
* beginner.jpg
* intermediate.jpg
* pro.jpg
* surfing.jpg

These images represent surfing lessons, training levels, and ocean surfing scenes.
