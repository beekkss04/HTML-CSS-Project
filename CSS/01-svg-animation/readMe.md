# SVG Shapes, Gradient and Animation

This project demonstrates the use of **SVG (Scalable Vector Graphics)** in HTML.
It includes different shapes, gradient colors, and basic animations.

---

## Project Objective

The objective of this project is to understand how SVG works and how shapes and animations can be created directly inside HTML without using images.

---

## Features

* Circle shape
* Rectangle shape
* Line drawing
* Text inside SVG
* Gradient color fill
* SVG animation
* Simple layout using CSS

---

## SVG Elements Used

The project uses the following SVG tags:

* `<svg>`
* `<circle>`
* `<rect>`
* `<line>`
* `<text>`
* `<defs>`
* `<linearGradient>`
* `<animate>`

These elements help create vector graphics and animations inside the browser.

---

## Animation Example

The circle animation changes its radius continuously:

```
<animate attributeName="r" values="30;45;30" dur="2s" repeatCount="indefinite"/>
```

This makes the circle expand and shrink repeatedly.
