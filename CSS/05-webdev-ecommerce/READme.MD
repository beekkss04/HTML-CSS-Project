# ShopEasy – Ecommerce Website 🛒

This project is a simple ecommerce-style website built using HTML and CSS.
It displays electronic products with prices and a clean shopping layout.

---

## Project Objective

The objective of this project is to create a basic online shopping interface and understand how ecommerce websites are structured using HTML and CSS.

---

## Features

* Ecommerce style header
* Search bar
* Navigation links
* Sale banner / hero section
* Product grid layout
* Product cards with image, rating and price
* Categories section
* Newsletter subscription section
* Footer

---

## Technologies Used

* HTML5
* CSS (Internal Styling)

---

## Products Displayed

The website currently displays:

* Wireless Headphones
* Smart Watch
* Gaming Mouse
* Bluetooth Speaker

Each product includes:

* Image
* Name
* Rating
* Price
* Add to Cart button (UI only)

---

## Folder Structure

```
├── index.html
└── README.md
```

Images are loaded using external links.

---

## How to Run

1. Open the project folder.
2. Double-click **index.html**.
3. The ecommerce website will open in your browser.
